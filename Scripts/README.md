# This is the game Droppin
## How To
Move the Ball sideways with your keyboard arrows or acceleratometer or device touch and through the holes.  
You get Points when you fall further down.  
The Extras empower you or make the game harder.  

# Development
## Setup
Install Godot and get startet. The Project is mainly written in gdscript. Use your own export templates.
